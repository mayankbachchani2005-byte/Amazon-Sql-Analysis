# Amazon Sales Analysis Using SQL

This project analyzes Amazon product sales data using SQL queries.  
The goal is to extract meaningful insights from the dataset by performing different SQL operations such as filtering, searching, and analyzing product and review data.

## Project Description

Amazon is one of the world's largest e-commerce platforms. In this project, SQL is used to analyze product information such as prices, discounts, categories, and customer reviews.

By using SQL queries, we can filter and analyze the data to support business decisions and understand product trends.

## Tools Used

- MySQL / SQL Server
- Dataset containing Amazon product details

## Database Structure

The table used in this project contains the following columns:

- product_id
- product_name
- category
- discounted_price
- actual_price
- discount_percentage
- rating
- rating_count
- about_product
- user_id
- user_name
- review_id
- review_title
- review_content
- img_link
- product_link

## SQL Queries Performed

1. List all products with a discounted price below ₹500.
2. Find products with a discount percentage of 50% or more.
3. Retrieve products where the name contains the word "Cable".
4. Display the difference between the average actual price and discounted price.
5. Find reviews that mention "Fast Charging".
6. Identify products with discount percentage between 20% and 40%.
7. Find products with actual price above ₹1000 and rating ≥ 4.
8. Find products where the discounted price ends with 9.
9. Display reviews containing negative words such as worst, waste, poor, or not good.
10. List products where the category includes "Accessories".

## Purpose of the Project

The purpose of this project is to practice SQL queries and learn how to analyze real-world e-commerce data.

It demonstrates how SQL can be used for:

- Data filtering
- Data analysis
- Business insights
- Customer review analysis
